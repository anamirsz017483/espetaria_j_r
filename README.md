# Espetaria J&R

Site de pedidos para espetaria com:
- capa com cadastro/login
- foto de fundo na capa
- cardápio com imagem de cada espeto ao lado
- carrinho com total automático
- pagamento por débito, pix, dinheiro e crédito
- envio do pedido pelo WhatsApp
- botão de cancelar pedido que volta para a capa

## Como usar
1. Abra `index.html` no navegador.
2. O WhatsApp já está configurado para o número **15991322438**.
3. Faça seu cadastro na capa e monte o pedido.

## Observação
As imagens dos espetos foram criadas como artes ilustrativas em SVG para o site ficar pronto no ZIP.
